# atlas-forge
Scripts used for building new templates and atlases


The Atlas Forge relies heavily on "BrainGlobe Template Builder". See the [BrainGlobe Template Builder](https://brainglobe.info/brainglobe-template-builder/) docs for details on installation and input specification.
